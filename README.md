## CocoaPods:

* For iOS8+: 

```ruby
use_frameworks!
target '<Your Target Name>' do
    pod 'CMPageTitleView'
end
```

* For iOS7+:

```ruby
target '<Your Target Name>' do
	pod 'CMPageTitleView'
end
```
