## <a id="installation"></a>Installation 

### CocoaPods:

* For iOS8+: 

```ruby
use_frameworks!
target '<Your Target Name>' do
    pod 'CMPageTitleView'
end
```

### Manually:
* Drag all files under `CMPageTitleView/CMPageTitleView/Class` folder into your project.
